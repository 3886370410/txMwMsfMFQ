```markdown
# 项目简介

本项目是一款基于Python开发的鞋类用户评价情感分析及可视化AI智能问答系统，旨在帮助用户更好地理解用户评价，并实现智能问答。

## 核心功能
- 登录注册
- 数据概览
- 情感分析
- 关键词分析
- 地域分布分析
- 时间趋势分析
- 评价详情查询
- 智能问答
- 用户管理
- 系统反馈

## 适用场景
- 鞋类产品制造商
- 鞋类零售商
- 鞋类电商平台
- 鞋类消费者

## 技术特点
- 使用Django作为后端框架
- 使用MySQL进行数据存储
- 使用Vue进行前端开发
- 使用ECharts进行数据可视化
- 基于NLP情感分析技术进行用户评价情感分析
- 使用TF-IDF算法进行关键词提取
```
